Тема: Створення основних класів у проєкті
Мета: Навчитися проєктувати та реалізовувати класи з використанням ООП

Опис проєкту
Консольна програма "Система Університету" на Python. Реалізує базову логіку роботи зі студентами, викладачами та курсами.

Класи:

Person (абстрактний) – ім'я, вік, адреса

Student – номер студентського, оцінки, розрахунок GPA

Instructor – зарплата, департамент

Course – назва курсу, код, список студентів, викладач

Як запустити:

Відкрити термінал у папці з проєктом

Виконати: python main.py
