# Лабораторна робота 23: Наслідування та поліморфізм

**Тема:** Наслідування та поліморфізм в об'єктно-орієнтованому програмуванні  
**Мета:** Навчитися використовувати наслідування та поліморфізм  

## Опис проєкту
Система управління працівниками компанії. Демонструє принципи наслідування та поліморфізму.

## Класи

| Клас | Базовий клас | Опис |
|------|--------------|------|
| `Employee` | ABC (abstract) | Абстрактний базовий клас |
| `Developer` | Employee | Розробник з бонусом за досвід |
| `Manager` | Employee | Менеджер з бонусами за проєкти та команду |
| `Intern` | Employee | Стажер з 50% зарплати |
