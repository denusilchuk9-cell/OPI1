# Лабораторна робота 24: Тестування ПЗ

**Тема:** Тестування та аналіз якості програмного забезпечення  
**Мета:** Навчитися проводити тестування програмного продукту  


## Структура проєкту

- `test_plan.md` - тестовий план
- `test_employee_system.py` - модульні тести
- `load_test.py` - навантажувальні тести
- `security_test.py` - тести безпеки
- `test_report.md` - звіт про тестування
- `employee_fixed.py` - покращена версія

## Як запустити

# Встановити залежності
pip install psutil

# Запустити всі тести
python -m unittest test_employee_system.py

# Запустити навантажувальні тести
python load_test.py
