Практичне заняття

Створення простого веб-сайту HTML та CSS

Мета заняття: навчитися створювати структуру сторінки, додавати контент, стилізувати елементи та адаптувати сайт для різних пристроїв.

Переглянути завдання До домашнього

Що ми вивчаємо

HTML

Створення структури сторінки: заголовки, абзаци, списки, зображення та посилання.

CSS

Оформлення сайту за допомогою кольорів, шрифтів, відступів, блоків та анімацій.

Responsive

Адаптація макета під мобільні пристрої, планшети й настільні комп'ютери.

Основні завдання

  1. Створити HTML-документ з елементами <head>, <body> і <footer>.
  2. Додати текст, зображення, посилання, заголовки, абзаци та списки.
  3. Підключити зовнішній файл стилів та стилізувати елементи сторінки.
  4. Оформити блоки за допомогою Flexbox і налаштувати відступи та розміри.
  5. Зробити адаптивний дизайн для різних екранів.
  6. Створити меню навігації та виділити активну сторінку.
  7. Перевірити код, знайти й виправити можливі помилки.
  8. Підготувати коротку презентацію з демонстрацією результату.

Приклади HTML-елементів

На цій сторінці використані різні теги: заголовки, абзаци, списки, кнопки-посилання, зображення та блоки.

  • Маркірований список для коротких пунктів
  • Нумерований список для послідовних кроків
  • Посилання на іншу сторінку сайту
