ПРЕЗЕНТАЦІЯ: Обробка виключень у Python

1. ОСНОВНІ КОНЦЕПЦІЇ:
- try: блок для потенційно небезпечного коду
- except: обробка конкретних винятків
- else: виконується при відсутності помилок
- finally: завжди виконується (очищення ресурсів)

2. ВЛАСНІ ВИКЛЮЧЕННЯ:
- Наслідування від класу Exception
- Контекстна інформація у повідомленнях
- Ієрархія власних класів помилок

3. ПРАКТИЧНІ ПІДХОДИ:
- Специфічні except перед загальними
- Логування помилок для налагодження
- Повторні спроби для тимчасових помилок
- Валідація вхідних даних

4. ПРИКЛАДИ З КАЛЬКУЛЯТОРА:
- ZeroDivisionError: ділення на нуль
- ValueError: некоректне число
- InvalidOperatorError: власна помилка
- NumberOutOfRangeError: власна помилка діапазону

5. ПЕРЕВАГИ:
- Надійність програм
- Зрозумілі повідомлення про помилки
- Коректне закриття ресурсів
- Гнучкість у обробці різних сценаріїв
